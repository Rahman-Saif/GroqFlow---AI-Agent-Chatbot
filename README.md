# ⚡ GroqFlow - AI Agent Chatbot

GroqFlow is an AI-powered agent chatbot built with a modern full-stack architecture.  
It combines the power of **Groq Cloud LLMs**, **LangGraph agent workflows**, and **Tavily web search** to provide intelligent, context-aware responses.

The project includes a **FastAPI backend** for AI processing and a **Streamlit frontend** for an interactive chat experience.

## 🚀 Features

- 🤖 AI Agent powered chatbot
- ⚡ Fast inference using Groq Cloud
- 🧠 LangGraph-based agent workflow
- 🔥 Llama model integration
- 🌐 Real-time web search capability with Tavily
- 🏗️ Full-stack architecture (Frontend + Backend)
- 💬 Interactive chat interface
- 🔌 REST API communication using FastAPI
